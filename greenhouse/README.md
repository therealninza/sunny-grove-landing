# Greenhouse Project

This folder contains information about the Bitcoin Greenhouse project's web application and related resources.

## Original Repository

The greenhouse web application is maintained in a separate repository:
**[https://github.com/asoltys/greenhouse](https://github.com/asoltys/greenhouse)**

This is a SvelteKit-based web application that was originally built to showcase the Bitcoin Greenhouse project.

## About the Project

The Bitcoin Greenhouse uses Bitcoin mining heat to create sustainable growing environments. This repository contains documentation and resources related to the physical build, while the web application lives in the linked repository above.

## Getting Started

To work with the greenhouse web application:

1. Clone the repository:
   ```bash
   git clone https://github.com/asoltys/greenhouse.git
   cd greenhouse
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

## Technologies Used

- **SvelteKit** - Web framework
- **Tailwind CSS** - Styling
- **Vercel** - Deployment

## Contributing

To contribute to the greenhouse web application, please visit the [original repository](https://github.com/asoltys/greenhouse) and submit pull requests there.

For hardware designs, schematics, and physical build documentation, please add those resources to this folder in the sunny-grove-landing repository.
